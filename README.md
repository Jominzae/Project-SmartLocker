# Project-SmartLocker

## 프로젝트 소개
STM32 마이크로컨트롤러를 기반으로 한 스마트 도어락 시스템 구현

STM32F411 보드, RFID, 4x4 Key, 서보 모터, LCD를 활용
<br>

## 개발 기간
* 25.08.06 - 25.08.08

## 개발 환경
- `STM32`

## 사용 기술
- 키패드 비밀번호 인증
- RFID 카드 인증
- LCD 디스플레이 출력
- 서보 모터 제어를 통한 문 잠금/해제
